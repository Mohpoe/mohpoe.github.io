Persamaan Kuadrat

Memahami Bentuk Umum dan Mencari Akar-akar Persamaan

1. Bentuk Umum

Persamaan kuadrat adalah persamaan polonimial derajat dua. Bentuk umumnya adalah:

ax² + bx + c = 0

Dengan syarat: a tidak boleh sama dengan 0.

2. Cara Faktorisasi

Salah satu cara mencari akar (nilai x) adalah dengan faktorisasi. Jika a = 1, kita mencari dua angka yang:

  • Jika dijumlahkan hasilnya adalah b
  • Jika dikali hasilnya adalah c

Rumus Faktorisasi:

(x + p)(x + q) = 0

Maka x = -p atau x = -q

3. Contoh Soal

Selesaikan: x² + 5x + 6 = 0

1. Cari angka yang jika dikali = 6, dan dijumlah = 5.

2. Angka tersebut adalah 2 dan 3 (karena 2×3=6 dan 2+3=5).

3. Bentuk faktor: (x + 2)(x + 3) = 0

4. Akar-akar: x + 2 = 0 → x = -2 atau x + 3 = 0 → x = -3

Himpunan Penyelesaian: {-3, -2}

4. Latihan Mandiri (15 Soal)

*Petunjuk: Ketik angka saja, lalu tekan tombol +/- untuk mengubah tanda negatif.
